# System Prompt: 極簡倒數計時器 (簡化版)

## 🎯 專案目標
建立一個單檔案 (HTML/CSS/JS) 的極簡倒數計時器，強調現代感設計 (Glassmorphism) 與強大的 URL 狀態共享能力。

## 🎨 視覺規範
- **風格**：毛玻璃效果 (backdrop-filter: blur(20px))。
- **進度環**：SVG 圓形進度環，支援動態 `stroke-dashoffset`。
- **主題系統**：
  - **白天**：紫色漸層背景，白色文字，**紫色進度環 (#a855f7)**。
  - **黑夜**：深色漸層背景，淺灰色文字，藍色進度環 (#60a5fa)。
- **版面**：
  - Header 使用 `grid-template-columns: 40px 1fr 40px` 確保標題絕對置中。
  - 響應式設計：手機版面自動調整間距與按鈕堆疊。

## ⚙️ 核心功能
1. **時間調整**：時、分、秒獨立 +/- 控制；5/10/15/20/30 分鐘預設按鈕。
2. **音效系統**：使用 Web Audio API 生成音效 (點擊/結束鬧鈴)，不使用外部音檔。
3. **快捷鍵**：Space (開始/暫停), ↑/↓ (調整分鐘), R (重設)。
4. **本地儲存**：自動記憶最後時間設定、主題與自訂標題。

## 🔗 URL 狀態共享 (核心邏輯)
URL 必須包含完整狀態，允許複製網址後在任何裝置「繼續」計時：
- **參數**：
  - `t`: 初始總秒數 (固定值，不隨倒數變動)。
  - `s`: 狀態 (`running` 或 `paused`)。
  - `start`: Unix 時間戳記 (秒)，表示該次計時起始的時間點。
  - `title`: 自訂標題 (URL 編碼)。
- **邏輯**：
  - 計算剩餘時間：`current = t - (now - start)`。
  - 載入時若狀態為 `running` 且時間未結束，自動啟動計時器。

## 🛠 技術規格
- **技術棧**：原生 HTML5, CSS3 (Variables), JavaScript (ES6+)。
- **效能**：單檔案 < 25KB，零外部依賴。
- **精度**：基於系統時間戳記計算剩餘時間，避免瀏覽器休眠導致的計時誤差。
